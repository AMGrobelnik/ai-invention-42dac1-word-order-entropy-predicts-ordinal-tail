## Full Summary

This dataset provides a per-language morphological complexity index from Grambank v1.0.3 mapped to Universal Dependencies treebanks via ISO 639-3. The pipeline downloads Grambank CLDF data from GitHub, grambank-analysed feature domain classification, and Glottolog CLDF for ISO mapping. It identifies 87 morphology-relevant features using authoritative domain classification (nominal/verbal/pronoun domains crossed with argument marking, TAME, number, class groupings) plus known morphological codes. For each language, it computes a complexity index (proportion of present morphological features out of coded ones, min 15-feature threshold). Languages map to UD treebanks via Glottocode-to-ISO through Glottolog, with macro-language resolution (arb→ara, cmn→zho). Output: 2,427 examples — 106 overlapping Grambank+UD (fold="overlap"), 2,253 Grambank-only, 68 UD-only. Each example: input=ISO code, output=JSON with morph index and individual feature values, plus metadata for glottocode, language name, family, macroarea, coordinates, and UD treebanks. Index ranges 0.08 (isolating, e.g. Mandarin) to 0.56 (inflected), mean 0.33. Provenance: Grambank v1.0.3 (Skirgard et al. 2023, CC-BY-4.0), Glottolog CLDF (CC-BY-4.0), grambank-analysed feature grouping.
